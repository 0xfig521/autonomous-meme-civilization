# Pitch Materials / 路演材料

## English Pitch

### 15-second Pitch

Autonomous Meme Civilization turns memes into living onchain civilizations on Sui. Users create a faction from a prompt, then autonomous AI-style agents recruit followers, spread propaganda, form alliances, start meme wars, and persist key identity and history as Sui objects.

### 30-second Pitch

Most meme products are static: a symbol, a chart, and a community chat. Autonomous Meme Civilization makes memes behave like living political species. Each civilization has lore, personality, mood, stats, community pressure, rivals, and onchain provenance. The arena runs in realtime, so judges and audiences can watch civilizations evolve, fight, collapse, and climb the leaderboard.

### 60-second Pitch

Autonomous Meme Civilization is an AI-native civilization simulator for Sui Overflow 2026. A user enters a prompt like "a frog cult that worships green candles." The app generates a civilization identity, starts an autonomous behavior loop, and places the faction into a realtime world with leaderboards, event feeds, community voting, and meme wars.

The core idea is that meme communities should not be static assets. They can be autonomous social organisms. Sui provides the object layer: `MemeCivilization`, `CivilizationEvent`, `Alliance`, and `MemeWar` objects make the world state persistent and composable. Walrus can store the larger lore, memory, and generated media. The result is a consumer-facing Agentic Web experience where memes become living onchain species.
